# 🏥 Hospital Management System

A desktop Hospital Management System developed using **Java**, **JavaFX**, and **SQLite**.

## Features

- Patient Management
- Doctor Management
- Nurse Management
- Appointment Management
- Medical Records
- SQLite Database Integration
- JavaFX Graphical User Interface (GUI)

## Technologies Used

- Java
- JavaFX
- SQLite
- NetBeans

## Project Structure

```
Hospital_system
│
├── src/
├── nbproject/
├── test/
├── hospital.db
├── build.xml
└── manifest.mf
```

## How to Run

1. Clone the repository.
2. Open the project in NetBeans.
3. Build the project.
4. Run the main class (`GuiMain.java`).
